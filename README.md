An AI platform that turns text prompts into functional websites instantly using a modern tech stack.

## Features
- AI Builder: Text-to-website generation via OpenAI.
- Full-Stack: React (Vite) frontend and Node.js (Express) backend.
- Auth & DB: Secure sessions via Better-Auth and database management with Prisma.
- Payments: Integrated Stripe subscription support.

## Quick Start

### Backend
1. cd server
2. npm install
3. npx prisma generate
4. npm run server

### Frontend
1. cd client
2. npm install
3. npm run dev

## Structure
- client/: React UI and site editor.
- server/: API, Auth, and AI logic.
